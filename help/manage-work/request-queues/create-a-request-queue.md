## 轮到你了

让我们花点时间回顾一下您刚才介绍的信息。

**问题：** 请求队列必须满足以下哪一项条件才能启用？ （选择所有适用项）

1. 项目状态=当前
1. 已选择请求类型中的“问题”
1. 已选择“作为帮助请求队列发布”
1. 项目状态=当前或计划

**回答：** 1)项目状态=当前和3)已选中“作为帮助请求队列发布”

**问题：** 可以通过电子邮件提出请求。 是真的还是假的？

**回答：** True

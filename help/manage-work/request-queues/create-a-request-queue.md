# 在Workfront中创建请求队列

该视频提供了有关在Workfont中创建请求队列的分步指南。 本&#x200B;视频的结尾演示了如何使用新创建的队列提交请求，显示了在“提交的请求”选项卡中输入详细信息并验证提交的过程。  

>[!VIDEO](https://video.tv.adobe.com/v/335221/?quality=12&learn=on&enablevpops=1)

## 关键要点

* **创建请求队列：**&#x200B;首先要创建一个新项目并命名该项目，该项目稍后将用作请求队列。 请&#x200B;确保选中“发布为帮助请求队列”框以正式将其指定为队列。  
* **访问权限：**&#x200B;通过从四个选项中进行选择，定义可向队列提交请求的人员：任何人、项目查看者、公司成员或组成员。  
* **请求类型：**&#x200B;自定义队列以包含特定类型的请求，如问题、错误报告、更改单或常规请求。 默认选项&#x200B;为“问题”。  
* **可自定义的字段：**&#x200B;设置其他详细信息（如默认响应时间、权限和其他队列特性）以定制其功能。  
* **使队列处于活动状态：**&#x200B;保存设置并将项目状态设置为“当前”以激活队列。 通过&#x200B;提交请求并在“提交的请求”选项卡中验证来测试其功能。

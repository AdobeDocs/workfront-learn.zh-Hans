# 查找和管理请求

在此视频中，您将会了解如何查找分配给您的请求、您创建的请求以及您管理的请求。

您将会了解到如何在以下位置找到这些请求：

* Workfront [!UICONTROL Home]
* [!UICONTROL Requests] 区域
* [!UICONTROL Teams] 页面
* 在自定义仪表板中

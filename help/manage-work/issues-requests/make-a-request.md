# 提出请求

在许多组织中，启动新项目、开发新产品或创建交付项的第一步是在以下位置提出请求： [!DNL Workfront]. 您可以跟踪请求的进度，并在需要时提供其他信息。

在本视频中，您将了解如何：

* 导航到请求区域
* 提出请求
* 查看提交的请求
* 查找请求的草稿

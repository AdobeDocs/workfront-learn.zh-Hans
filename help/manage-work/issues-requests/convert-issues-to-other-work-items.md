# 将问题/请求转化为任务

在本视频中，您将了解如何将问题或请求转化为任务。

>[!VIDEO](https://video.tv.adobe.com/v/3427605/?quality=12&learn=on)

>[!NOTE]
>
>您不能将问题添加到项目时间表中，因为它们代表“规划外的工作”。项目时间线用于“规划的工作”，即任务。

# 创建工作角色和费率卡

此视频介绍如何创建工作角色和费率卡，以及如何从项目附加和删除费率卡。


>[!VIDEO](https://video.tv.adobe.com/v/3486528/?captions=chi_hans&quality=12&learn=on&enablevpops=1)

## 关键要点

* **工作角色结构项目任务：**&#x200B;工作角色表示任务所需的技能，可以根据用户的能力和可用性分配给用户。
* **费率卡定义记帐费率：**&#x200B;费率卡列出工作角色的记帐费率，并充当项目记帐的与客户合同。
* **灵活的费率管理：**&#x200B;工作角色和费率卡都允许使用有效日期设置成本和记帐费率，支持基于时间的定价更改。
* **费率卡覆盖：**&#x200B;将费率卡附加到项目时，其费率会覆盖工作角色费率，但项目特定的覆盖不受影响。
* **轻松的附件和更新：**&#x200B;可以从项目中附加或删除费率卡，并且可以将更新推送到现有项目以确保帐单准确无误。

# 浏览项目管理的过滤器、视图和分组

此视频概述了Workfront中的筛选器、视图和分组，以帮助用户有效地管理和分析项目、任务和问题。 要点&#x200B;包括：

* **项目筛选器：**&#x200B;筛选器可自定义以缩小项目列表范围。 &#x200B;示例包括“我拥有的项目”、“我参与的项目”、“当前”和“落后于预定计划”。 对“处于风险中”和“落后于预定计划”等&#x200B;过滤器进行了详细解释，说明它们如何依赖不同的字段。
* **项目视图：**&#x200B;视图显示各种项目详细信息，如计划、预计和实际开始/完成日期。 例&#x200B;如“标准视图”、“所有日期视图”和“真实故事视图”，其中提供条件更新和摘要。  
* **项目分组：**&#x200B;项目可以按进度状态或完成日期等字段按月分组，从而更好地进行组织和分析。  
* **任务筛选器：**&#x200B;筛选器可帮助识别与特定角色相关的未分配任务或任务。 清除筛选器&#x200B;会显示父级任务结构，排序可按任务编号调整。  
* **任务视图：**&#x200B;诸如“状态视图”之类的视图显示任务状态和进度、注释、文档、问题、审批、关键路径和里程碑的标志。
* **任务分组：**&#x200B;与项目分组类似，任务可以按进度状态等字段分组。  
* **问题：**&#x200B;问题的筛选器、视图和分组与项目和任务的相似。 受欢迎&#x200B;的视图包括“登入信息”，以及“主要联系人”等分组可帮助组织问题。  

该视频强调自定义，并提供指向教程的链接以供进一步学习。

## 关键要点

* **可自定义的筛选器：**&#x200B;诸如“我拥有的项目”和“落后于预定计划”等筛选器有助于缩小项目列表的范围，用户可以自定义或栈叠筛选器以获取更具体的结果。  
* **详细视图：**&#x200B;视图（如“所有日期视图”和“真实故事视图”）提供了对计划、预计和实际日期的分析，以及项目摘要和条件更新。  
* **高效分组：**&#x200B;可以按进度状态或完成日期等字段对项目和任务进行分组，从而更好地进行组织和分析。  
* **任务管理：**&#x200B;任务筛选器和视图（如“状态视图”）可帮助识别未分配任务、跟踪进度以及访问相关详细信息（如注释、文档和里程碑）。  
* **问题跟踪：**&#x200B;过滤、查看和分组问题，如“条目信息”和“主要联系人”，简化问题管理并明确提交时间表和联系人。

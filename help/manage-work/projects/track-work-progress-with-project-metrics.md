# 使用项目指标跟踪工作进度

此视频介绍如何使用项目量度跟踪项目进度。 它&#x200B;演示了访问项目左侧面板中的量度选项卡，可能需要滚动并单击“显示更多”。 顶部的KPI区域显示关键绩效指标，例如已完成、未完成和超期的任务。 用户&#x200B;可以单击指示器查看特定任务，并使用图表按状态或优先级对任务或问题进行分组。 将鼠标悬停&#x200B;在列上可显示数量，图表显示了完成与未完成任务或问题的比率。 量度还可以&#x200B;按被分派人根据其状态（完成、未完成、即将发生或过期）显示任务。 会&#x200B;保存查看量度的偏好设置以供将来访问。 最&#x200B;后，本视频演示了如何使用导出按钮将“量度”页面导出为PNG文件。 


>[!VIDEO](https://video.tv.adobe.com/v/336667/?quality=12&learn=on&enablevpops=1)

## 关键要点

* **访问项目量度：**&#x200B;导航到项目左侧面板中的“量度”选项卡以跟踪进度。 您&#x200B;可能需要向下滚动，然后单击“显示更多”以找到它。 
* **关键绩效指标(KPI)：** KPI区域显示已完成、未完成和超期的任务。 单&#x200B;击指示器可让您查看特定任务。 
* **任务和问题图表：**&#x200B;按状态或优先级对任务或问题进行分组，显示完成与未完成的比率。 将鼠标悬停&#x200B;在列上可显示数量。 
* **按被分派人列出的量度：**&#x200B;按被分派人根据其状态（完成、未完成、即将执行或过期）查看任务。 会&#x200B;保存首选项以供将来访问。 
* **导出量度：**&#x200B;通过单击屏幕右上角的“导出”按钮，将量度页面导出为PNG文件。 &#x200B;

# 直接从模板创建项目

此视频介绍如何从模板创建项目。 如&#x200B;果您有权打开模板，则可以将其用作创建项目的快捷方式。 为此&#x200B;，请打开模板，访问三个圆点菜单，然后选择“创建项目”。 这&#x200B;会将您转到可编辑项目详细信息的屏幕。 准备就绪&#x200B;后，单击“创建项目”以完成。  

>[!VIDEO](https://video.tv.adobe.com/v/3456013/?quality=12&learn=on&enablevpops=1)

## 关键要点

* **模板访问：**&#x200B;您需要权限才能打开模板以从中创建项目。  
* **快捷方式选项：**&#x200B;与从头开始项目相比，使用模板是一种方便的快捷方式。  
* **三点菜单：**&#x200B;打开模板，然后从三点菜单中选择“创建项目”。  
* **编辑项目详细信息：**&#x200B;选择“创建项目”后，您可以在最终确定之前编辑项目详细信息。  
* **完成创建：**&#x200B;当您准备好完成该过程时，请单击“创建项目”。

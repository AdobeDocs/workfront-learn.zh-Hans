# 为创建布局模板 [!DNL Workfront Goals] 用户

在此视频中，您将学习如何：

* 创建布局模板
* 将布局模板分配给用户
* 添加 [!DNL Goals] 到布局模板
* 更改 [!DNL Goals] 符合贵组织的术语
* 添加 [!DNL Goals] 到 [!DNL Workfront] [!UICONTROL Main Menu]

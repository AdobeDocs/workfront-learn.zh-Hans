# 为 [!DNL Workfront Goals] 用户创建布局模板

在 **[!UICONTROL Setup]** 区域内，您还可以创建和分配包含 [!DNL Workfront Goals] 的布局模板，以便您和您的用户可以更轻松地访问它。

在本视频中，您将学习如何：

* 创建布局模板
* 将布局模板分配给用户
* 将 [!DNL Goals] 添加到布局模板
* 根据您的组织的需要更改 [!DNL Goals] 术语
* 将 [!DNL Goals] 添加到 [!DNL Workfront] [!UICONTROL Main Menu]

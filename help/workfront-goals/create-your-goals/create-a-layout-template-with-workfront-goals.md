# 使用创建布局模板 [!DNL Workfront Goals]

在 **[!UICONTROL Setup]** 区域，您还可以创建和分配布局模板，其中包括 [!DNL Workfront Goals] 这样您和您的用户便可以更轻松地访问该页面。

在此视频中，您将学习如何：

* 创建布局模板
* 将布局模板分配给用户
* 添加 [!DNL Goals] 到布局模板
* 更改 [!DNL Goals] 符合贵组织的术语
* 添加 [!DNL Goals] 到 [!DNL Workfront] [!UICONTROL Main Menu]

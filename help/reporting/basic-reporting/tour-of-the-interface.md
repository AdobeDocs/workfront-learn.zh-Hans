# 浏览日历报表界面

日程表报告是一种动态报告，其中会在日程表视图中显示工作和休息时间。 您可以在日程表报告中显示以下对象的日期信息：

* 任务
* 项目
* 问题
* 休息时间

在本视频中，您将了解到：

* 日程表分组
* 日程表视图
* 日程表操作菜单
* 概要信息
* 以及如何创建临时日程表项目

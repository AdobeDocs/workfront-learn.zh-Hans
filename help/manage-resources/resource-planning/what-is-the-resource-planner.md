# 什么是 [!DNL Resource Planner]?

为了真正增强人的权能，管理者必须规划和安排其人的努力。 但是，在尝试捕获这些数据并按月、按季、甚至按年管理数据的同时，找到具备适当技能的人员，这是一项重大任务。

但是，使用Workfront的资源规划器，这项任务会稍微容易一些。

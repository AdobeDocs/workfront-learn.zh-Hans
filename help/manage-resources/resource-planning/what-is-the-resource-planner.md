# 什么是 [!DNL Resource Planner]？

要真正增强人们的权能，管理者必须计划和安排员工的工作。 然而，在适当的时间找到具有适当技能的人员，同时尝试捕获此数据并每月、每季度甚至每年管理它，是一项重大任务。

但是，使用Workfront的资源规划器执行这项任务会容易一些。

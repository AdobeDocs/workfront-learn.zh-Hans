# 什么是 [!DNL Resource Planner]？

为了能够真正地为员工提供支持，管理人员必须对员工的工作进行规划和安排。但是，在试图获取这些数据并在每月、每季度甚至每年对其进行管理的同时，找到在正确的时间有空并且具备正确技能的人员是一项重大的任务。

但是，使用 Workfront 的资源规划程序可以使这项任务变得更容易一些。

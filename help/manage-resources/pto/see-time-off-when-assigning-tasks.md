# 在分配任务时查看空闲时间

小图标表示在您分配用户期间，用户在个人日历上标记了空闲时间。

通过键入用户名从项目计划分配时，您会看到其名称灰显，并在其头像图片旁边显示一个小型蓝色警告图标。

![pto的用户灰显](assets/toat_01.png)

如果您从任务页眉进行分配，您会在该人员的头像旁边看到一个小型蓝色飞机图标，提醒您该人员在此任务持续期间有空闲时间。

![用户任务分配](assets/toat_02.png)

# 通用连接器演练简介

## 概述

使用电子表格中的宠物小精灵角色，通过 HTTP 连接器调用 Poke API 来收集并发布有关该角色的更多信息。

练习网址：`https://pokeapi.co/api/v2/pokemon/{Character}`

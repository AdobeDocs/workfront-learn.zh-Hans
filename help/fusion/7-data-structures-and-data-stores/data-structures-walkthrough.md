# 数据结构演练

打开包含时间条目列表的 CSV 文件。 这些时间条目是多个用户在某些天记录的分钟数。 我们的目标是使用此信息生成一个新的 CSV，其中显示每个用户每天记录的总时间（以小时为单位）。

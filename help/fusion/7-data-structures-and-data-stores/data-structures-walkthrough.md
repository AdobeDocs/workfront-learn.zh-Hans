# 数据结构演练

## 概述

打开包含时间条目列表的CSV文件。 这些时间条目是指多个用户在特定日期内记录的分钟数。 其目标是获取此信息并生成一个新的CSV，以显示每个用户每天记录的总时间（以小时为单位）。

# 创建多阶段校样工作流

校样工作流程非常简单，只需要几个审阅人和一个截止日期。 或者，它们可能会更加复杂，因为有多个截止日期，而且人们会按特定顺序查看证据。

[!DNL Workfront] 具有两种类型的校样工作流 — 基本和自动工作流，有时称为高级。

自动化工作流具有多个阶段或步骤，这些阶段或步骤需要按特定顺序完成。 每个阶段除了自己的截止时间之外，还有其自己的一组收件人（审阅人和批准人）。

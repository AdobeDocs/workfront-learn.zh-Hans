# 创建多阶段验证工作流

验证工作流可以很简单，其中只需要几个审核人和一个截止日期。或者它们也可以更复杂，其中有多个截止日期，并且人们会按特定顺序查看该验证。

[!DNL Workfront] 有两种类型的验证工作流——基本和自动化（有时称为高级）。

自动化工作流具有多个阶段或步骤，这些步骤需要按特定顺序完成。每个阶段除了有自己的截止日期外，还有自己的一组收件人（审核人和审批人）。

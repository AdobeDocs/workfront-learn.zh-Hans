# 校对用户的设置

在中开始创建用户之前 [!DNL  Workfront]，最好让您的全球 [!DNL Workfront] 和校样系统设置已完成。 其中某些设置在创建用户配置文件时会应用于用户配置文件。 但是，如果您在创建某些用户之前未获取这些集，则无需担心……您可以编辑每个用户配置文件中的这些设置。


在此视频中，您将学习如何：

* 确定哪些用户获得校样许可证
* 在 [!DNL  Workfront]
* 在校样后端设置中调整校样用户设置

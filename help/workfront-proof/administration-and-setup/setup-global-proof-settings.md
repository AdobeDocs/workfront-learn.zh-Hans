# 设置全局验证设置

通过建立可影响用户的全局默认设置，启动 [!DNL Workfront] 中的验证系统设置。现在设置这些默认值可确保您创建的用户拥有开始使用验证功能所需的最低设置...此外，它还可以节省您完成验证设置的时间。

* 分配默认验证角色
* 建立默认帐户设置
* 设置默认验证设置

如果您在建立这些默认值之前已经开始创建用户，请不要担心。您可以在各个用户配置文件上编辑所有这些信息。

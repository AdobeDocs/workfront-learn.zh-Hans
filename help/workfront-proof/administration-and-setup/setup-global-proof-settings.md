# 设置全局校对设置

在中启动系统设置以进行校对 [!DNL Workfront] 建立影响用户的全局默认设置。 现在，设置这些默认值可确保您创建的用户具有开始使用校对功能所需的最低设置……此外，它可在您完成校对设置时节省您的时间。

* 分配默认验证角色
* 建立默认帐户设置
* 设置默认校对设置

如果在建立这些默认设置之前已开始创建用户，请不要担心。 您可以在各个用户配置文件中编辑所有这些信息。

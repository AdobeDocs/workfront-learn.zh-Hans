# 通过拖放上传

您可以使用拖放功能直接在 [!UICONTROL Documents] 部分上传 [!DNL Workfront] 中的文件。 这是一种快速上传校样的方法，但不允许您同时分配校样工作流。

![[!DNL  Workfront] 项目中的 [!UICONTROL Documents] 区域的图像，其中光标悬停在文档列表上，并且 [!UICONTROL Drag & Drop documents here] 消息可见。](assets/drag-and-drop-1.png)

对于一些 [!DNL Workfront] 客户，这是他们常规的校样工作流。 创意设计师或团队成员会上传文件，但需要由校对管理者或项目经理来分配校对工作流，以便相应的收件人可以审阅工作。

根据您的 [!DNL Workfront] 系统设置，通过拖放上传时可能不会生成校样。 如果是这种情况，您需要使用 [!UICONTROL Generate Proof] 选项并选择 [!UICONTROL Advanced] 生成校样并分配工作流。 从那里，您可以设置您的校样工作流。

![突出显示 [!UICONTROL Generate Proof] 的 [!DNL  Workfront] 项目中 [!UICONTROL Documents] 区域的图像。](assets/drag-and-drop-2.png)

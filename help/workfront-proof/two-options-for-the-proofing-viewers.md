# 有关校对观众的解释

[!DNL Workfront] 允许您查看两个主要类别的校对——静态的和交互式的。

静态校样包括 PDF、图像和文本文件等。 一些网站校样属于交互式校样类别，具体取决于校样的上传方式。

[!DNL Workfront]根据您正在查看的校对类型（以及校对系统管理员进行的一些设置），打开相应版本的校对查看器（网络校对查看器或桌面校对查看器）。

网络校对查看器是 [!DNL Workfront’s] 对静态校对和视频的默认选择。 当您在电子邮件或 [!DNL Workfront] 中单击 [!UICONTROL Open Proof] 时，网络查看器将会作为 [!DNL Workfront] 浏览器窗口的一部分打开。

桌面校对查看器是一个在您的工作站上运行的独立应用程序。 它主要用于使用 URL、嵌入式或流媒体视频、交互式横幅和 HTML5 动画创建的交互式网站校样。 该查看器允许您单击网站的链接和菜单并导航到不同的页面。

这两个查看器的工作方式相同，因此无论校对类型如何，都可以确保协作审阅。

## 安装桌面校对查看器

第一次打开包含交互式内容的校对时，系统会提示您下载桌面校对查看器。 只需按照说明操作即可。

如果您的组织不允许您在工作站上安装应用程序，请与系统管理员合作，确保您拥有所需的校对查看器。

安装桌面校对查看器应用程序后，您的交互式校对将来就会在该查看器中打开。 您只需下载一次即可。

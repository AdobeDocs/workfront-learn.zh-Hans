# 规划项目 - 审查

使用项目模板快速启动和运行您的 [!DNL  Workfront] 项目。然后，首先通过关注这五个领域来规划您的项目：

1. 任务
1. 持续时间
1. 规划小时数
1. 前置任务
1. 分配

一切设置完毕后，将项目状态更改为“当前”以启动项目。随着项目的进展，您需要跟踪完成的工作、出现的问题、需要回答的问题等等。请记住——您可以在 [!DNL Workfront] 中查找和管理所有这些信息！

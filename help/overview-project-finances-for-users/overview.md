# 用户项目财务概述

了解如何跟踪项目的财务信息、设置成本和收入类型以及覆盖记帐费率。

Workfront并不是要取代您当前使用的任何财务跟踪或账单软件，但它可以提供货币跟踪和报表。

>[!PREREQUISITES]
>
>* 对使用Workfront的基本了解
>* 有权编辑财务数据的标准许可证类型

## 目标受众

希望在Workfront中跟踪财务的用户。
